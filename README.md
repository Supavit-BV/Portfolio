# Supavit Portfolio

A modern, interactive portfolio website for Supavit, showcasing AI-driven culinary innovation and automation services.

## About

Supavit combines passion for flavor with intelligent automation, offering AI-enhanced blends, automated culinary strategy consulting, and flavor innovation workshops. This portfolio site presents our story, services, and collections in an elegant, interactive format.

## Features

- **Responsive Design**: Optimized for all devices and screen sizes
- **Interactive Navigation**: Smooth scrolling between sections with active state indicators
- **Dynamic Mouse Effects**: Custom CSS glow effects that follow cursor movement
- **Modern UI**: Clean, professional design with custom styling
- **Fast Performance**: Built with Vite for optimal loading speeds

## Technologies Used

- **React 18** - Modern JavaScript library for building user interfaces
- **Vite** - Fast build tool and development server
- **CSS3** - Custom styling with modern features and animations
- **JavaScript ES6+** - Modern JavaScript with hooks and state management

## Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server:
```bash
npm run dev
```

The site will be available at `http://localhost:5173`

### Build for Production

Build the project for production:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```
